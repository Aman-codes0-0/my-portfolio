# Aman Kumar's Portfolio 🚀

A modern, responsive, and aesthetically pleasing personal portfolio website built with HTML, CSS, and JavaScript. It features a unique **Neon-Glassmorphism** design, smooth micro-animations, and dynamic interactions.

## 🌟 Live Demo
[Check out the Live Portfolio Here](https://Aman-codes0-0.github.io/my-portfolio) *(If you have deployed this repo)*

## ✨ Key Features
- **Neon-Glassmorphism Theme:** A sleek and premium UI design with an interactive dark/light mode toggle.
- **Interactive Cards:** Beautifully animated project and achievement cards that pop out individually on hover.
- **Functional Contact Form:** Integrated with EmailJS to receive messages directly without a backend server.
- **Time Machine Feature:** A creative floating button linking back to legacy/older portfolio versions.
- **Fully Responsive:** Perfectly optimized layout for both desktop and mobile views.

## 🛠️ Technologies Used
- **HTML5:** Semantic and accessible structure.
- **CSS3:** Vanilla CSS, Grid, Flexbox, custom variables, and keyframe animations.
- **JavaScript:** Vanilla DOM manipulation and theme state management.
- **External:** FontAwesome Icons & Google Fonts (Inter, Outfit).

## 📁 Repository Structure
- `index.html` - Home Page & Hero Section
- `projects.html` - Showcasing all recent development projects
- `achievements.html` - Certifications and Hackathon recognitions (e.g., SIH 2025)
- `resume.html` - Professional journey timeline and core skills
- `contact.html` - Integrated EmailJS messaging form
- `style.css` - Global styles, animations, and responsive media queries
- `source/` - Directory containing images and other static assets
